Echelon Form Generator

Instraction

In the following box you can put all the coefficients and constants of all linear equations. Here a set of equation and input procedure for the box are given. After putting the input values click on 'Generate' button.

Sample Equations
x1 +2x2 +3x3 = 4
2x1 +3x2 +4x3 = 5
3x1 +4x2 +5x3 = 6

Input Format
1 2 3 4
2 3 4 5
3 4 5 6
